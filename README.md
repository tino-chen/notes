## 目录结构

```
notes/
├── llm/        # LLM 相关：训练、推理、提示工程等
├── agent/      # Agent 相关：架构、框架、设计模式等
├── tools/      # 工具使用：OpenClaw、开发工具等
├── projects/   # 项目相关知识
└── misc/       # 杂项
```

## 文档规范

- **命名**：kebab-case，如 `prompt-engineering.md`
- **格式**：
  ```markdown
  # 标题
  > 来源：xxx | 日期：YYYY-MM-DD | 标签：#tag1 #tag2
  
  内容...
  ```
